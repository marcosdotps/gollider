# Gollider

## Overview

Gollider is a collision detection library written in Go. It provides efficient algorithms for detecting collisions between geometric shapes in 2D space.

## Project Status

This project is in its early stages of development. The initial scaffolding and CI pipelines have been set up.

## Getting Started

### Prerequisites

- Go 1.24 or later

### Building

```bash
go build ./...
```

### Running

```bash
go run main.go
```
